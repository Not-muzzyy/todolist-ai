## 🚀 About The Project

This project was built during my **first-ever hackathon** — a full-stack web application that goes beyond a basic todo list by integrating AI features for smarter task management.

The goal was to build something practical, deployable, and genuinely useful — all within a tight hackathon timeframe.

-----

## ✨ Features

- ✅ **Add & manage tasks** — simple and intuitive UI
- ✅ **Mark tasks as completed** — track your progress
- ✅ **Delete tasks** — keep your list clean
- 🤖 **AI-powered suggestions** — smart task recommendations
- 🧠 **Smart organization** — intelligent task grouping
- 🎨 **Clean responsive UI** — works on all screen sizes
- ⚡ **Python backend** — fast and lightweight

## 📦 Installation & Usage

```bash
# 1. Clone the repository
git clone https://github.com/Not-muzzyy/todolist-ai.git
cd todolist-ai

# 2. Install backend dependencies
pip install -r requirements.txt

# 3. Start the backend server
cd Backend
python app.py

# 4. Open the frontend
# Open Frontend/index.html in your browser
```

## 💡 How the AI Works

The AI layer analyzes your task descriptions and:

- Suggests **priority levels** based on keywords
- Recommends **similar tasks** you might have missed
- Helps **organize tasks** into smart categories automatically
