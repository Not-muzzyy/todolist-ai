# TodoList-AI

A simple AI-powered To-Do List application.  
This project lets users add, manage, and track tasks, enhanced with automation or AI features (e.g., suggestions, smart organization). It’s a practical full-stack starter project you can build on.

---

## 🧠 Features

✔ Add new to-do items  
✔ Mark tasks as completed  
✔ Delete tasks  
✔ Smart/AI suggestions *(if implemented)*  
✔ Frontend + Backend structure

---

## 🧱 Tech Stack

- **Backend:** Python  
- **Frontend:** HTML, CSS, JavaScript  
- **Optional AI integration:** (e.g., OpenAI, GPT models)
